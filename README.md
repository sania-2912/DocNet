#Docnet

Docnet is a networking platform that connects doctors, nurses, hospitals, medical students, dealers, and suppliers in India. It aims to create a unified space for the medical sector, providing a centralized job board for hospitals to post vacancies and job seekers to apply directly.

#Features

•	Centralized Job Board: Hospitals can post job openings, and candidates can apply directly.
•	Networking: Doctors, nurses, medical students, and suppliers can connect on a single platform.
•	User Profiles: Create and manage professional accounts with personal and career-related information.
•	Search & Recommendations: Find job opportunities or medical professionals with the search bar and recommendations.
•	Mobile App: A mobile version of the platform is in development for easier access.

#Tech Stack

•	Frontend: HTML, CSS, JavaScript, React+Vite
•	Backend: Node.js, Express.js
•	Database: MySQL
